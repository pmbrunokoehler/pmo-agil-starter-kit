# Template — Fluxo de Intake

> Formulário padrão para capturar uma nova iniciativa e conduzi-la pelos estágios de triagem até a decisão. Mantenha curto: intake longo demais afasta as pessoas de registrar.

| Campo | Conteúdo |
|-------|----------|
| **Nome da iniciativa** | |
| **Solicitante / dono** | |
| **Data de entrada** | |
| **Problema que resolve** | (o problema, não a solução) |
| **Valor esperado** | (negócio, cliente, risco evitado) |
| **Alinhamento estratégico** | (a qual objetivo/OKR se conecta) |
| **Esforço estimado** | (faixa: baixo / médio / alto) |
| **Riscos e dependências conhecidos** | |
| **Prazo desejado e por quê** | |

## Estágios do intake

- [ ] **Captura** — registro feito no formato acima
- [ ] **Triagem** — faz sentido? alinhado? tem dono? → segue ou recusa
- [ ] **Qualificação** — detalhado o suficiente para priorizar
- [ ] **Decisão** — entra no backlog priorizado / fica em espera / recusado

## Boas práticas

- Recusar com justificativa é tão importante quanto aprovar. Uma recusa explicada preserva a relação e ensina o critério.
- Triagem deve ser rápida (minutos, não dias). Qualificação detalhada só para o que passou na triagem.
- Tudo que entra fica visível na mesma lista de portfólio. Iniciativa que não está na lista não existe para o PMO.
