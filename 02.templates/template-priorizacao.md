# Template — Canvas de Priorização de Portfólio

> Modelo enxuto para comparar iniciativas com um critério explícito. Escolha um dos métodos abaixo e use de forma consistente. A pontuação abre a discussão; não a encerra.

## Opção A — Matriz Valor × Esforço (a mais simples)

Posicione cada iniciativa nos quadrantes:

| | Baixo esforço | Alto esforço |
|--|--------------|--------------|
| **Alto valor** | 🟢 Fazer primeiro | 🟡 Planejar bem |
| **Baixo valor** | ⚪ Se sobrar tempo | 🔴 Evitar |

## Opção B — RICE

Pontuação = (Reach × Impact × Confidence) ÷ Effort

| Iniciativa | Reach | Impact (0.25-3) | Confidence (%) | Effort | Score RICE | Ranking |
|-----------|-------|-----------------|----------------|--------|------------|---------|
| [nome] | | | | | | |

- **Reach:** quantos serão impactados num período.
- **Impact:** quão forte o impacto (0.25 mínimo, 3 massivo).
- **Confidence:** quão confiante você está na estimativa.
- **Effort:** esforço total estimado (pessoa-mês, por exemplo).

## Opção C — Alinhamento estratégico ponderado

| Iniciativa | OKR 1 (peso X) | OKR 2 (peso Y) | OKR 3 (peso Z) | Score total | Ranking |
|-----------|----------------|----------------|----------------|-------------|---------|
| [nome] | | | | | |

Pontue a contribuição de cada iniciativa a cada objetivo (0-3) e multiplique pelo peso do objetivo.

## Regras de uso

- Use o mesmo método para todas as iniciativas do ciclo. Misturar critérios invalida a comparação.
- A pontuação é insumo para a conversa de priorização, não veredito automático.
- Registre a decisão final e o porquê, especialmente quando ela divergir da pontuação — isso documenta o julgamento.

> 📝 **[PREENCHER]:** mantenha apenas o método que você usa e remova os outros, para o template ficar enxuto.
